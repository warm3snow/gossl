## gossl
gossl is a crypto library implemented in golang. It provides a command-line tool to operate on data.

## Features
### Base Cryptography Primitives
#### Symmetric Cryptography
- [x] AES-256-CBC
- [x] SM2-CBC

#### Asymmetric Cryptography
- [x] RSA2048
- [x] ECDSA
- [x] SM2
- [ ] Ed25519

#### Hashing
- [x] SHA256
- [x] SHA3-384
- [x] SHA3-512
- [x] SM3
- [ ] HMAC

#### KDF
- [x] PBKDF2
- [x] BCRYPT
- [x] SCRYPT
- [x] Argon2

### TLS
- [x] TLS1.1
- [x] TLS1.2
- [x] TLS1.3
- [x] GMTLS1.1
